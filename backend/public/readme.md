# 🌤️ Weather Forecast App

A beautiful, real-time weather forecast application that provides current weather conditions for any city worldwide.

## ✨ Features

- 🔍 Search for any city globally
- 🌡️ Real-time temperature display
- 💧 Humidity information
- 🌬️ Wind speed data
- 🎨 Beautiful, responsive UI with animations
- 📱 Mobile-friendly design
- ⚡ Fast backend API with Express.js
- 🛡️ Comprehensive error handling

## 🛠️ Technologies Used

### Backend
- Node.js
- Express.js
- Axios
- CORS
- Dotenv

### Frontend
- HTML5
- CSS3 (with animations and gradients)
- Vanilla JavaScript (ES6+)
- WeatherAPI.com API

## 📋 Prerequisites

- Node.js (v14 or higher)
- npm (Node Package Manager)
- Git
- WeatherAPI.com API key (free)
