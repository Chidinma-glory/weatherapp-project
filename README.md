# 🌤️ Weather Forecast App

A beautiful weather app that shows real-time weather for any city.

## Features
- Search any city
- Shows temperature, humidity, wind speed
- Beautiful design with animations

## How to Run
1. Get API key from WeatherAPI.com
2. Add it to backend/.env file
3. Run `npm install` then `npm start`
4. Open index.html

## Technologies
- Node.js + Express backend
- HTML/CSS/JavaScript frontend
- WeatherAPI.com API
